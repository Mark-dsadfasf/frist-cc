# 番茄钟桌面应用

## 技术栈
- **前端**: React + TypeScript + Vite
- **后端**: Tauri 2.x (Rust)
- **状态管理**: React hooks
- **样式**: TailwindCSS

## 功能需求

### 核心功能
1. **番茄计时器**
   - 默认 25 分钟专注工作
   - 默认 5 分钟短休息
   - 默认 15 分钟长休息（每 4 个番茄后）
   - 可调整时长

2. **控制功能**
   - 开始 / 暂停
   - 重置
   - 跳过（进入下一个阶段）

3. **状态显示**
   - 倒计时显示（分:秒）
   - 当前阶段（专注/短休息/长休息）
   - 已完成番茄数量

4. **通知**
   - 计时结束系统通知
   - 桌面通知

5. **系统集成**
   - 窗口置顶（可选）
   - 最小化到系统托盘
   - 托盘菜单（显示/退出）

## 界面设计
- 简约现代设计
- 大号数字显示倒计时
- 进度条显示当前番茄进度
- 清晰的状态指示器
- 响应式布局
