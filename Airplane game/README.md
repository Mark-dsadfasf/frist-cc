# 飞机射击游戏 / Airplane Shooter

一款复古像素风格的卷轴射击街机游戏。

## 运行方式

直接用浏览器打开 `index.html` 即可开始游戏。

支持的浏览器：Chrome、Firefox、Edge、Safari 等现代浏览器。

## 操作说明

| 按键 | 功能 |
|------|------|
| 方向键 / WASD | 移动飞机 |
| 空格键 / Z | 发射子弹 |
| ENTER | 开始游戏 / 重新开始 |

### 触控支持

- 点击屏幕开始/重新开始
- 滑动屏幕移动飞机
- 手指按住屏幕持续射击

## 游戏玩法

- 操控你的战机，消灭从上方来的敌机
- 躲避敌方子弹和碰撞
- 击落敌机获得分数
- 收集掉落的强化道具提升火力（PWR）
- 生命值归零游戏结束

## 敌机类型

| 类型 | 外观 | 特点 | 分数 |
|------|------|------|------|
| 普通 | 橙色 | 直线下落 | 10 |
| 快速 | 红色小型 | 速度快 | 15 |
| 精英 | 紫色大型 | 会射击、HP=3 | 30 |

## 强化系统

击落敌机有概率掉落强化道具：
- PWR 1: 单发子弹
- PWR 2: 双发子弹
- PWR 3: 三发子弹（最强）

强化持续一段时间后会自动降级。

## 技术栈

- HTML5 Canvas
- Vanilla JavaScript
- Web Audio API（音效）
- CSS3

## 本地开发

无需任何构建工具，直接双击 `index.html` 即可运行。

如需修改代码：
1. 修改 `js/game.js` 调整游戏逻辑
2. 修改 `css/style.css` 调整样式
3. 修改完成后刷新浏览器即可看到效果

## 数据存储

游戏最高分保存在浏览器本地 `localStorage` 中。
